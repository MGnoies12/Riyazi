category

در ریاضیات، مجموعه به مجموعه‌ای از اشیاء مشخص و مجزا گفته می‌شود. این اشیاء می‌توانند هر چیزی باشند: اعداد، افراد، حروف، یا اشکال هندسی. اعضای یک مجموعه را درون آکولاد { } می‌نویسیم.

مثال: مجموعه اعداد زوج کوچکتر از ۱۰

A = {2, 4, 6, 8}

انواع مجموعه‌ها

مجموعه تهی (Empty Set)

مجموعه‌ای که هیچ عضوی ندارد و با نماد ∅ یا {} نمایش داده می‌شود.

B = {}

مجموعه متناهی (Finite Set)

مجموعه‌ای که تعداد اعضای آن قابل شمارش است.

C = {1, 2, 3, 4, 5}

مجموعه نامتناهی (Infinite Set)

مجموعه‌ای که تعداد اعضای آن قابل شمارش نیست.

D = {1, 2, 3, 4, ...}
